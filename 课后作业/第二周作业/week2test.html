CAT介绍

计算机辅助翻译（Computer aided translation，CAT）类似于CAD（计算机辅助设计），能够帮助翻译者优质、高效、轻松地完成翻译工作。 它不同于以往的机器翻译软件，不依赖于计算机的自动翻译，而是在人的参与下完成整个翻译过程。与人工翻译相比，质量相同或更好，翻译效率可提高一倍以上。CAT使得繁重的手工翻译流程自动化，并大幅度提高了翻译效率和翻译质量。计算机辅助翻译是一个广泛的和不精确的术语，涵盖了一系列的工具，从非常简单到更复杂的。

CAT软件-Trados

TRADOS，这一名称取自三个英语单词。它们分别是：Translation、Documentation和Software。其中，在“Translation”中取了“TRA”三个字母，在“Documentation”中取了“DO”两个字母，在“Software”中取了“S”一个字母。把这些字母组合起来就是“TRADOS”了。透过这三个英语单词的含义，我们可以想见“TRADOS”的取名还是很有用意的。因为这恰恰体现了TRADOS软件所要达到的功能和用途。

CAT软件-MemoQ

memoQ 软件包括： 译员专业版（TPro）、项目经理版（PM）、服务器版（Server）。

译员专业版
顾名思义是给译员用的，在高校就是给学生用的，译员专业版可以作为服务器版的桌面客户端，而且是配置最多的桌面客户端。每台初装电脑都可以直接从官方免费申请一个专业版试用许可，试用期45天，45天之后要么使用正版许可激活，要么就自动转为功能受限的译员免费版。
项目经理版
顾名思义是给项目经理用的，在高校可以给老师用，也可以给学生用，招标文件中说的教师端软件或教学版软件就是指项目经理版。 从翻译技术教学的实际出发，为了能够模拟翻译公司的项目流程，尤其是组织分组协作翻译，一般会建议5个译员专业版配1个项目经理版（1名项目经理、3-4位译员、1-2位审校），最多9个译员专业版配1个项目经理版（1名项目经理、5-7位译员、2-4位审校，其中一审1-2位，二审1-2位）。
服务器版
是翻译公司或高校采购的版本，服务器版软件专为项目管理和翻译协作定制，是名副其实的实时协作解决方案。可以实现人员管理、项目管理、进度管理、资源管理等。通过自动化流程和实时资源共享可以大大提高翻译效率，保证翻译的准确性和一致性。通过 API 还可以与其他平台对接，实现功能的扩展，具体请参看公众号前面发布的内容。

CAT软件-SmartCAT

Smartcat是一款基于云计算环境的软件，能够满足公司（包括翻译公司）和自由译员的翻译工作流程需求。因此，它的定位较为多元，既是一款计算机辅助翻译软件，又是全球化管理软件和外包平台。
